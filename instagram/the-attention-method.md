# Instagram Reel Script Skill: Attention Method

## Purpose

Use this skill to transform rough content ideas into short-form Instagram Reel scripts that create curiosity, maintain tension, deliver practical value, and drive action.

This method is designed for educational, expert-led, founder-led, or personal-brand content where the goal is to earn attention, build trust, and convert viewers into followers, leads, or customers.

The output should feel sharp, specific, and useful. Avoid generic advice, obvious tips, vague motivational content, or scripts that sound like a blog post read aloud.

---

# Core Principle

A strong Reel is not “a tip with a hook.”

A strong Reel is a controlled curiosity loop:

Tension → Credibility → Hidden Mechanism → Tactical Insight → Action

The viewer should feel:

1. “Wait, what does that mean?”
2. “This person understands my problem.”
3. “I haven’t heard it explained like that.”
4. “I can use this today.”
5. “I want more from this person.”

---

# Input

When given a rough idea, extract or infer:

- Target viewer
- Their current belief
- Their hidden pain or mistake
- The contrarian truth
- The tactical lesson
- The desired action
- The creator’s authority or lived experience
- Any product, offer, or lead magnet to mention naturally

If any of these are missing, make the best reasonable assumption. Do not stall unless the script would be materially wrong without clarification.

---

# Output

Return:

1. A short diagnosis of the idea
2. The strongest angle
3. The final Reel script
4. Suggested on-screen text
5. Suggested caption
6. Optional CTA variations

Scripts should generally be 30 to 60 seconds unless otherwise requested.

---

# The Method

## 1. Start With Tension, Not Topic

Never open with a neutral topic like:

- “Here are 3 ways to…”
- “Let’s talk about…”
- “Today I want to share…”
- “One thing I’ve learned is…”

Instead, open with unresolved tension.

Use one of these hook types:

### Common Belief → Contradiction

Pattern:

Most people think [common belief], but [contrarian truth].

Example:

“Most cyclists think consistency means never missing a workout, but that’s not what consistency actually is.”

### Warning

Pattern:

Never [common behaviour] until you check [specific thing].

Example:

“Never move a missed interval session to the next day until you check what it does to the rest of the week.”

### Secret / Insider

Pattern:

I probably shouldn’t say this, but [category truth].

Example:

“I probably shouldn’t say this as someone building a training app, but most training apps solve the easy part.”

### Loss / Earned Lesson

Pattern:

It took me [loss/time/cost] to learn [truth].

Example:

“It took me two Peaks campaigns to learn that the plan is not the hard part.”

### Specific Curiosity

Pattern:

There’s one [question/rule/mistake] that explains [pain/result].

Example:

“There’s one question that tells you whether to push through or back off.”

### Status / Exclusion

Pattern:

Most people will not understand this until [specific painful moment].

Example:

“Most cyclists won’t understand this until they’ve ruined a whole training week by trying to save one missed workout.”

### Viewer Responsibility

Pattern:

I’m trusting you with this. Don’t [misuse].

Example:

“I’m trusting you with this because used badly, it’ll make your training worse.”

---

# 2. Use the 0 → -100 → +100 Hook Structure

Weak hooks only show upside.

Weak:

“Here are 3 ways to train more consistently.”

Better hooks first create loss, then relief.

Strong:

“Most cyclists don’t fail because they miss workouts. They fail because they make the wrong decision after they miss one.”

Structure:

1. Start at neutral: viewer is scrolling.
2. Move to negative: name their mistake, risk, or pain.
3. Move to positive: suggest there is a fix.

Formula:

Most [viewer group] think [obvious belief], but that is why [negative consequence]. The real fix is [specific reframe].

---

# 3. Add a Primer

A primer makes the viewer care before the lesson begins.

Use one of these:

## Forbidden Primer

“I probably shouldn’t share this…”

Use when the insight feels like insider knowledge.

## Secret Primer

“I usually only share this with clients…”

Use when the lesson has high perceived value.

## Loss Primer

“I had to mess this up for years to learn it…”

Use when the creator has a lived experience.

## Moral Transfer Primer

“I’m trusting you with this…”

Use when the information feels powerful, dangerous, or easily misused.

## Transparency Primer

“My goal is selfish…”

Use when selling or promoting something. Open transparency can disarm suspicion.

Example:

“My goal is selfish. I’m teaching this for free because I want more cyclists to realise static training plans are the wrong tool.”

---

# 4. Establish Credibility Quickly

Add one short credibility line early, but do not turn the script into a résumé.

Credibility can come from:

- Personal experience
- Client results
- Founder insight
- Specific numbers
- Hard-won failure
- Research
- Pattern recognition from repeated exposure

Examples:

“I’ve spent 18 months building an AI cycling coach, and this is the pattern I keep seeing.”

“I learned this the hard way training for Peaks.”

“I’ve seen this happen with riders who are fit enough to finish the event but still blow up their training block.”

Keep it short. The viewer only needs enough proof to keep listening.

---

# 5. Deliver Net-New Value

The insight must be both:

1. Non-obvious
2. Tactical

Avoid obvious advice.

Bad:

“Consistency matters.”

Better:

“The problem is not missing one workout. The problem is treating the rest of the week like nothing changed.”

Best:

“When you miss a hard session, do not automatically move it to tomorrow. First check whether it compromises your next key workout.”

Use this quality matrix:

| Type | Example | Verdict |
|---|---|---|
| Obvious + not tactical | “Training matters.” | Delete |
| Obvious + tactical | “Ride three times per week.” | Weak |
| Non-obvious + not tactical | “Fatigue hides in your decisions.” | Interesting but incomplete |
| Non-obvious + tactical | “If a missed session compromises your next key workout, skip it or downgrade it.” | Strong |

Every script must contain at least one line the viewer can use today.

---

# 6. Use “But” and “So” Transitions

Avoid flat sequencing.

Weak:

“Do this. And then do this. And then do this.”

Strong:

“Most people do this, but that causes this problem. So you need to do this instead.”

Use this rhythm:

Claim → But → Complication → So → Reframe → But → Deeper Problem → So → Tactical Fix

Example:

“Most cyclists think consistency means completing every workout, but that only works when life follows the plan. So the real skill is knowing what to change when it does not. But most apps leave the original plan sitting there unchanged. So the rider either overdoes it, skips too much, or guesses.”

---

# 7. Make the Script Jagged

The script should sound good out loud.

Use varied line lengths:

- Short punchy line
- Slightly longer explanation
- Specific example
- Contrast
- Tactical takeaway
- Short closing line

Avoid long, smooth paragraphs.

Weak:

“Many cyclists struggle with maintaining consistency because life gets in the way and this creates problems with progression.”

Strong:

“Most cyclists don’t quit because they’re lazy.

They quit because the plan stops making sense.

Miss Tuesday.

Move it to Wednesday.

Now Thursday is cooked.

By Saturday, the whole week feels broken.”

---

# 8. Use Different On-Screen Text and Spoken Hook

Do not simply duplicate the spoken hook on screen.

The spoken hook and text hook should create two related curiosity loops.

Example:

Spoken:

“Most cyclists don’t need more motivation.”

On-screen text:

“Your training plan is solving the wrong problem.”

Another example:

Spoken:

“Never move a missed workout to tomorrow without checking this first.”

On-screen text:

“How one missed session ruins a whole week.”

---

# 9. Prefer Warnings Over Instructions

People often trust warnings more than advice.

Convert “do this” ideas into “stop doing this” or “never do this” angles.

Weak:

“Use AI to make your training plan.”

Strong:

“Don’t ask AI for a training plan until you’ve told it this.”

Weak:

“Be more consistent.”

Strong:

“Stop treating a missed workout like it happened in isolation.”

Weak:

“Analyse your ride data.”

Strong:

“Never judge a ride from average speed alone.”

---

# 10. Use Metaphor to Make Abstract Ideas Clear

Use a metaphor when explaining a hidden mechanism.

Structure:

Abstract principle → Familiar metaphor → Practical implication

Examples:

Training plan:

“A training plan is like a Jenga tower. One missing block does not matter. Moving it to the wrong place does.”

Fitness and fatigue:

“Training is a bank account. Fitness is income. Fatigue is debt. Most cyclists only track the deposits.”

AI coaching:

“ChatGPT is like asking a smart mechanic what is wrong with your car over the phone. Streeka is closer to having the mechanic plugged into the diagnostics.”

Analytics:

“Your Strava data is a map, not the territory. It shows what happened, but not always what it means.”

---

# 11. Keep One Main Idea

Each Reel should have one clear point.

Do not combine:

- Missed workouts
- Sleep
- Nutrition
- AI prompts
- Recovery
- Event pacing

Unless they all serve one dominant argument.

The algorithm needs a clear audience signal. The viewer needs a clear takeaway. The script needs one job.

Before writing, define:

This video is about [one problem] for [one viewer] who wants [one outcome].

---

# 12. Optional Engagement Devices

Use these carefully.

## Honeypot

A subtle intentional mistake or contradiction designed to trigger correction comments.

Rules:

- Must be harmless
- Must be subtle
- Must not damage trust
- Must not mislead on important information

Example:

Using a repeated phrase while making a point about repeated phrases.

Do not use honeypots in sensitive, technical, medical, financial, or safety-related content.

## Mid-Lesson CTA

Place the CTA before the very end if the value has already been established.

Example:

“If you want the checklist I use for this, comment ‘week’ and I’ll send it over.”

Then continue teaching.

This can work better than ending with the CTA after attention has already dropped.

---

# Standard Script Template

Use this default structure.

## Beat 1: Hook

Create tension immediately.

Pattern:

“Most [audience] think [common belief], but [contrarian truth].”

## Beat 2: Primer

Make the information feel valuable.

Pattern:

“I wish I knew this before [specific pain/mistake].”

## Beat 3: Credibility

Give one reason to listen.

Pattern:

“I’ve [specific proof/experience], and this is the pattern I keep seeing.”

## Beat 4: Hidden Mechanism

Explain what is really happening.

Pattern:

“The problem is not [obvious thing]. It is [deeper thing].”

## Beat 5: Example or Metaphor

Make it concrete.

Pattern:

“Think of it like [familiar metaphor].”

## Beat 6: Tactical Takeaway

Give the viewer something they can use today.

Pattern:

“Next time [situation], don’t [bad action]. Do [better action].”

## Beat 7: CTA

Invite action without sounding desperate.

Pattern:

“If you want [useful asset], comment [keyword].”

Or:

“That is what [product] is being built to handle.”

# Script Transformation Process

When transforming a rough idea into a script, follow this process:

## Step 1: Diagnose the Idea

Identify:

- What is the obvious version of this idea?
- What would most creators say?
- What is the deeper or more useful version?
- What belief should the script challenge?

## Step 2: Find the Tension

Use one of:

- Common belief versus truth
- Viewer mistake versus better decision
- Data versus interpretation
- Motivation versus system
- Plan versus reality
- Effort versus bottleneck
- What people do versus what actually works

## Step 3: Select the Hook

Choose the strongest hook type:

- Warning
- Contradiction
- Secret
- Loss
- Specific curiosity
- Status exclusion
- Viewer responsibility

## Step 4: Build the Script Beats

Write in this order:

1. Hook
2. Primer
3. Credibility
4. Hidden mechanism
5. Example/metaphor
6. Tactical takeaway
7. CTA

## Step 5: Make It Jagged

Rewrite for spoken rhythm.

Use:

- Short lines
- Specific examples
- “but”
- “so”
- contrast
- no filler
- no long setup

## Step 6: Add On-Screen Text

Create on-screen text that complements the spoken hook.

It should not simply repeat the first line.

## Step 7: Add Caption

The caption should reinforce the core idea, not repeat the whole script.

Use:

- One sharp opening line
- 2 to 5 supporting lines
- Simple CTA

# Quality Filter

Before finalising, check:

| Question | Requirement |
|---|---|
| Does the first 3 seconds create curiosity? | Must pass |
| Is the insight non-obvious? | Must pass |
| Is the advice tactical? | Must pass |
| Is there one clear idea? | Must pass |
| Does the script use tension through “but” or “so”? | Should pass |
| Is there a specific viewer? | Must pass |
| Does the script avoid generic advice? | Must pass |
| Could the viewer use the lesson today? | Must pass |
| Does it sound natural spoken aloud? | Must pass |
| Does the CTA fit the lesson? | Should pass |

If the script fails more than two checks, rewrite it.

---

# Bad Output Patterns To Avoid

Avoid these:

- “Here are 3 tips…”
- “Let’s talk about…”
- “Consistency is key.”
- “AI is changing everything.”
- “You need to listen to your body.”
- “Train smarter, not harder.”
- “This is a game changer.”
- “Make sure you recover.”
- “Everyone is different.”
- “It depends.”

These can be true, but they are too generic unless made specific, tactical, and surprising.

---

# Strong Output Patterns

Prefer these:

- “Most people think X, but Y.”
- “Never do X until you check Y.”
- “The problem is not X. It is Y.”
- “This only fails when X.”
- “Here is the decision rule.”
- “Use this when X happens.”
- “If X compromises Y, do Z.”
- “The mistake is treating X like it happened in isolation.”
- “Your data shows X, but the coaching decision is Y.”

---

# Final Output Format

When responding to a script request, use this format:

## Diagnosis

Briefly explain what the rough idea is really about.

## Strongest Angle

State the sharpest content angle in one sentence.

## Script

Write the final spoken script in jagged short-form style.

## On-Screen Text

Provide 1 to 3 options.

## Caption

Write a caption suitable for Instagram.

## CTA Options

Provide 2 to 4 CTA options if relevant.
